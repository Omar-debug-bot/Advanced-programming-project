# FreshMart – Online Supermarket

A full-stack online supermarket web application with product browsing, 
shopping cart, user accounts, and order management, built with Java and Spring Boot.

## Tech Stack
- Java 17
- Spring Boot 3.2
- Spring Data JPA
- H2 Database (in-memory, no installation required)
- Thymeleaf (frontend templating)
- HTML/CSS

## Features
- Product catalog with admin management (add/edit products)
- User registration, login, and authentication
- Shopping cart functionality
- Order placement and order history
- Admin dashboard for managing products, users, and orders
